*cat /etc/os-release* — Checks the Linux distribution version.

*top / htop* — Monitors live CPU, RAM, and process activity.

*df -h* — Checks available hard drive space.

*pwd* — Shows your current folder location.

*free -h* — Checks available system RAM memory.

*history* — Shows the list of commands you previously typed.

*ls* — Lists files and folders in your current location.

*mkdir* — Creates a new folder.

*clear* — Cleans the terminal screen.

*cd* — Changes your current directory/folder.

*echo* "text" > file.txt — Writes text directly into a file.

*cat* — Displays the entire contents of a file.

*apt update* — Refreshes the list of available software downloads.

*apt install -y vim* — Installs the Vim text editor utility.

*vim* — Opens a file to read or edit text.

*head / tail* — Shows lines from the very top or bottom of a file.

*date* — Displays the current system date and time.

*uptime* — Shows how long the system has been running.

*which* — Locates the exact path of a program file.

*uname -a* — Displays the core operating system kernel details.
